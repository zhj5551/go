## 下载
```
使用https://github.com/FelisCatus/SwitchyOmega/releases地址，下载SwitchyOmega_Chromium.crx文件到本地目录
把SwitchyOmega_Chromium.crx后缀修改成压缩包格式的后缀，比如SwitchyOmega_Chromium.rar
把刚才的压缩文件解压到当前目录
```
## 安装
```
在google浏览器中输入chrome://extensions/地址，并打开开发者模式
点击“加载已解压的拓展文件"，选择刚才解压的目录，会自动安装
```
## 使用
```
打开SwitchyOmega页面后，选择情景模式-proxy
在代理服务器的代理协议中选择socks5,在输入代理的IP地址与端口
点击”应用选项‘,使配置生效
```
